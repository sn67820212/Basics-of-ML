# Basics of ML

A comprehensive learning repository that takes you from the fundamentals of PyTorch to understanding basic ML codes that use autodifferentiation.

## 📚 Overview

This repository is designed for learners who want to:
- Understand the basics of **PyTorch** framework
- Learn how **autodifferentiation** works in machine learning
- Build foundational knowledge to read and understand basic ML code
- Practice implementing simple machine learning concepts

## 🎯 What You'll Learn

- PyTorch fundamentals and tensor operations
- Automatic differentiation (autodiff) concepts
- Building neural networks from scratch
- Gradient computation and backpropagation
- Basic machine learning algorithms

## 📋 Prerequisites

- Python 3.7 or higher
- Basic understanding of linear algebra and calculus
- Familiarity with Python programming

## 🚀 Getting Started

### Installation

1. Clone the repository:
```bash
git clone https://github.com/sn67820212/Basics-of-ML.git
cd Basics-of-ML
```

2. Install required dependencies:
```bash
pip install -r requirements.txt
```

### Quick Start

Explore the notebooks and scripts in the repository to get started with PyTorch and autodifferentiation.
